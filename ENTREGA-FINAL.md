## 🛠️ **Deploy Hostinger - 5 Passos:**

### 1. **Banco de Dados**

- Criar banco MySQL no cPanel
- Executar script `setup.sql` no phpMyAdmin

### 2. **Upload dos Arquivos**

- Extrair ZIP na pasta `public_html`
- Editar credenciais no `api/analytics.php`

### 3. **Configurar SSL**

- Ativar certificado SSL no cPanel
- Forçar redirecionamento HTTPS

### 4. **Testar Funcionamento**

- Acessar site em produção
- Testar formulário de inscrição
- Verificar dados no banco

### 5. **Monitoramento**

- Analytics automático funcionando
- Relatórios disponíveis via API
