## ⚠️ 6. POLÍTICA DE ALTERAÇÕES

### 📋 **DURANTE O DESENVOLVIMENTO**

#### 🟢 **PEQUENAS** (sem custo adicional)

- Correção de textos/ortografia
- Ajustes de cores dentro da paleta
- Pequenos ajustes de espaçamento
- Correção de bugs

#### 🟡 **MÉDIAS** (20-50% do valor da funcionalidade)

- Alteração de layout de uma seção
- Adição de campo em formulário
- Mudança de integração similar
- Nova página simples

#### 🔴 **GRANDES** (novo orçamento)

- Mudança completa de design
- Nova funcionalidade não prevista
- Integração com sistema externo
- Refatoração de código significativa

### 📋 **APÓS A ENTREGA**

#### 🟢 **1º MÊS** (suporte incluído)

- Correção de bugs
- Pequenos ajustes de conteúdo
- Suporte técnico básico
- Treinamento para uso

#### 🟡 **APÓS 1º MÊS**

```
Hora de desenvolvimento: R$ 120 - R$ 180
Alteração de conteúdo: R$ 80 - R$ 150
Suporte técnico: R$ 100 - R$ 200
```

## 🎯 9. DICAS COMERCIAIS

### ✅ **SEMPRE INCLUIR NO ORÇAMENTO**

1. Cronograma detalhado
2. Forma de pagamento clara
3. O que está e NÃO está incluído
4. Política de alterações
5. Prazo de validade
6. Garantias oferecidas

### ⚠️ **CUIDADOS ESPECIAIS**

- Sempre validar se cliente tem material necessário
- Confirmar se orçamento está alinhado com expectativa
- Deixar claro limite de alterações
- Definir responsável pela aprovação
- Documentar TUDO por escrito

### 🚀 **PARA FECHAR MAIS VENDAS**

1. Oferecer 3 opções (básica, intermediária, premium)
2. Destacar valor entregue vs concorrência
3. Mostrar cases similares (como Wake & Move)
4. Oferecer garantia de satisfação
5. Criar senso de urgência (desconto por tempo limitado)
