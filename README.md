# Wake & Move Primavera - Landing Page

Landing page oficial do evento "Wake & Move Primavera" - uma experiência única de corrida, yoga e bem-estar em Resende, RJ.

## 🌟 Sobre o Evento

O Wake & Move Primavera é mais que um evento, é uma comunidade saudável que promove hábitos transformadores através de:

- **Corrida**: Opções de 4km e 8km para todos os níveis
- **Yoga**: Prática de relaxamento e conexão
- **Palestras**: Bate-papo com profissionais de saúde
- **Café da Manhã**: Confraternização pós-atividades

## 📅 Informações do Evento

- **Data**: 26 de Outubro de 2025
- **Horário**: 7h
- **Local**: Aeroporto de Resende, RJ
- **Participantes**: +300 esperados

## 🚀 Tecnologias Utilizadas

- **React 18** com TypeScript
- **Vite** para build e desenvolvimento
- **Tailwind CSS** para estilização
- **Lucide React** para ícones
- **Mercado Pago SDK** para pagamentos

## 🛠️ Como executar

1. Instale as dependências:

```bash
npm install
```

2. Execute o projeto em modo de desenvolvimento:

```bash
npm run dev
```

3. Abra [http://localhost:5173](http://localhost:5173) no navegador

## 📦 Build para produção

```bash
npm run build
```
